# Brief - VFX/3D tab

## Goal
Create a dense, curated VFX/3D archive page for Matteo Ruggiero / sxilence that can work for a commercial studio opportunity while still feeling connected to the main site. It should show a strong visual world: weird in a good way, image-led, but with enough project-specific text to give context.

## User flow
1. Visitor opens the new `vfx/3d` tab from the homepage navigation.
2. Visitor sees a strong image wall, concise title, and dense index strip.
3. Visitor scans project cards with preview images, roles, credits, short project-specific text, stills, and source links.
4. Visitor reaches commercial/product work, archive works, and random GFX studies as connected layers.

## Mood / visual direction
Dark production archive, shot breakdown without becoming sterile. Keep it connected to the main broken archive: black glass, hard white lines, video stills, 3D fragments, credit blocks, dense visual rhythm, short grounded text.

## States to handle
- Hover/focus on project cards and source links.
- Responsive collapse from multi-column production grid to single-column mobile.
- Missing detailed credits: mark as selected role notes instead of pretending final credits are complete.

## Responsive behavior
- Desktop: hero split with image wall and lead copy, then dense 12-column project grid.
- Mobile: single-column hero, stable image ratios, readable credits, no text overlap.

## Non-goals
- Do not make a clean corporate reel page.
- Do not explain the work too much or flatten it into a dry reel.
- Do not over-feature Bastl; keep it as commercial/product proof but lower hierarchy.
